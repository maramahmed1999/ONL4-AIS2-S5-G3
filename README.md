# Excavator Activity Monitor System

A real-time computer vision system that monitors excavator activity from video streams using **YOLOv26n**, **ByteTrack**, and **Optical Flow**. The system detects and tracks excavators, classifies their activity as **WORKING** or **IDLE**, streams events through **Apache Kafka**, and visualizes live analytics in an interactive **Streamlit dashboard**.

## 🚀 Live Demo

**🔗 Live Application:**  
(https://equipmonitorsystem.streamlit.app/) 

> **Note:** The online demo is intended for exploring the dashboard interface only. Live monitoring is unavailable because the application relies on **Apache Kafka** events generated by the local computer vision pipeline. Running the complete system requires a local environment with **Docker** (for Kafka) and the CV service.

## Features
- Real-time excavator detection using YOLOv26n
- object tracking using ByteTrack
- Activity classification (WORKING / IDLE) using Farneback Optical Flow
- Live dashboard with metrics and charts
- Kafka-based event streaming
- Excel export for activity summaries
- YOLO performance monitoring (Confidence, FPS, Inference Time)
- Hard-frame collection for low-confidence detections

## Installation

Create virtual environment:
```bash
python -m venv .venv
```
Activate the environment:

```bash
# Windows
.venv\Scripts\activate

# Linux / macOS
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r src/requirements.txt
```

---

## Run

Start Kafka:

```bash
docker compose -f src/docker-compose.yml up -d
```

Launch the dashboard:

```bash
streamlit run src/dashboard/app.py
```

Open:

```
http://localhost:8501
```

The dashboard can process either:

- Uploaded videos
- Live camera streams
